* Gun
* Grenade launcher
* Rocket Launcher
* Power punch


### Implemeted Abilities
* Charge Boost
* Power Jump


### Map Editor Controls
* 1 - Save map
* 0 - Reset rotation
* B - Size mode
* C - Clone current block
* M - Mass mode
* N - New block
* P - Position Mode
* R - Rotation Mode
* G - Toggle physics/gravity
* U - Undo move
* X - Remove block
* Arrows & Pg Up/Pg Dn: Adjust model in dimensions depending on mode

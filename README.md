## Controls
* Mouse/keyboard
* Tested with PS4 controllers


### Implemeted Abilities
* Charge Boost


### Map Editor Controls:-

* 1 - Save map
* B - Size mode
* C - Clone current block
* N - New block
* P - Position Mode
* R - Rotation Mode
* Arrows & Pg Up/Pg Dn: Adjust model in dimensions depending on mode
* T - Toggle physics
* X - Remove block
